# Bank Cards Management System (REST API)

Ushbu loyiha bank kartalarini boshqarish, foydalanuvchilar o'rtasida pul o'tkazmalari va tranzaksiyalar tarixini kuzatish uchun mo'ljallangan Spring Boot backend ilovasi.

## 🚀 Texnologiyalar
* **Java**: 25 (Eclipse Temurin)
* **Framework**: Spring Boot 4.0.5
* **Database**: PostgreSQL 15
* **Migration**: Liquibase
* **Containerization**: Docker & Docker Compose
* **Security**: Spring Security & JWT
